# OpenWeatherAPI
Just a simple app for testing the [OpenWeatherAPI](https://openweathermap.org/)
Only the 'Current weather' and 'Five day forecast' calls are used, for a fixed location (London).

**Tech stack used**:
- Android Studio
- Linux
- Kotlin
- MVVM
- Dagger-Hilt
- Retrofit
- Coroutines
